Descrição do Projeto

Motivação: Treinar conceitos de lógica digital, temporização e I/O usando um jogo educativo.

Objetivo: Simular uma bomba com contagem regressiva e permitir que o jogador "desarme" inserindo o código correto (no hardware, via keypad).

Trecho de Código (Arduino)

// exemplo simplificado
#include <Keypad.h>
#include <LiquidCrystal_I2C.h>

LiquidCrystal_I2C lcd(0x27, 16, 2);
const int buzzer = 8;
int tempo = 60; // segundos

void setup(){
  lcd.init();
  lcd.backlight();
  pinMode(buzzer, OUTPUT);
}

void loop(){
  for(int t = tempo; t >= 0; t--){
    lcd.setCursor(0,0);
    lcd.print("Tempo: "); lcd.print(t);
    delay(1000);
    // checar teclado para código correto (omiti)
  }
  tone(buzzer, 1000);
  delay(2000);
  noTone(buzzer);
  while(1);
}

Problemas enfrentados e lições aprendidas

  • Debounce no keypad: ler repetido sem tratamento.
  • Endereço I2C diferente entre módulos de LCD.
  • Importância de testar cada componente isoladamente.
